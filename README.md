# tisa heal

> A 60-second guided anxiety-relief tool — type what's bothering you and watch it dissolve.

Built with React + Framer Motion + Tailwind CSS + Vite.

## Deploy to Vercel (one command)

```bash
npx vercel --prod
```

Or connect your GitHub repo on [vercel.com](https://vercel.com) — Vercel auto-detects Vite and sets everything up.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- **React 18** — component architecture
- **Framer Motion** — all animations and transitions
- **Tailwind CSS** — utility styling
- **Vite** — lightning-fast build tool
- **Web Audio API** — generative ambient drone + bell tones (no audio files needed)
